O que é o RLG - Real Life Gaming?

O RLG é um jogo que tem por objetivo promover a realização de tarefas que de outra forma seriam desinteressantes para o jogador, evitando hábitos sedentários, promovendo atividades físicas, entre outros.

Como vai ser desenvolvido o RLG?

O RLG vai ter por Core a linguagem Java, tendo por complemente o motor de jogo Unity.

O que podemos esperar do RLG?

Uma aplicação revolucionária que trará benifícios para todo o utilizador que decidir utilizar no seu dia a dia.

Quais as Features disponíveis?

Uma diversidade de tarefas que irão ser propostas ao utilizador, juntamente com um sistema de recompensa baseado em EXP e atribuições especiais com as nossas parcerias. Estará ainda disponível um sistema de Leaderboard para vermos quem é o utilizador que mais se está a empenhar.
